# CLI-Based To-Do List Application

## Description
A simple command-line based to-do list application built using Python.
It allows users to add, view, complete, and delete tasks using file storage.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using file I/O

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Run `python todo.py`
